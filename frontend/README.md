## Description

URL Shortener Client based on [Vite](https://vitejs.dev/)

## UI
![](/frontend/homepage.png "Homepage")

## Installation

```bash
$ yarn
```

## Running the app

```bash
$ yarn dev
```

Open [http://localhost:4000](http://localhost:4000) to view it in the browser.

## Bundle React

```bash
$ yarn build
```
