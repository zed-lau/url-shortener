## Description
Url Shortener Server based on [Nest](https://github.com/nestjs/nest) framework

## Swagger
![](/backend/swagger.png)

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# test coverage
$ yarn run test:cov
```
